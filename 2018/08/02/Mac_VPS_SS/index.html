Mac 上远程连接VPS 搭建SS

前言

最近在更新github博客时，每次更新博客提交Git都很费时间，最后忍无可忍，想办法做个能加速网络的东东。网上搜罗了一下，发现个神器叫vultr，可以使用Mac基于Vultr搭建ss来加速访问网络。
Vultr是一家美国的服务器厂商，这个VPS服务商是按小时收费的。这意味着，架设一台VPS的成本几乎为零。假如你不想用美国的了(这里推荐使用美国的，应为我试了5个IP，只有美国NL的IP可以玩，2333)，关机销毁然后再换到全球任何一个地区无压力。新手入门强烈推荐。

1). 点击首页打开注册，点击页面右上角的Create Account注册按钮即可注册。

2).注册好账号并登陆进去，点击加号开启VPS

3).选择一个你喜欢的服务器地区，推荐日本或者美国。

4).选择服务器系统，推荐选择Ubuntu 16.04。因为我们mac vultr搭建ss的教程是基于Ubuntu系统开始的。

5).选择套餐，笔者这里选第二个5刀每月的。2.5刀的配置不包括公共IPv4地址，部署后可以附加一个保留的IPv4地址，所以这里选择5刀的，因为5刀的流量更多，性能更高，而且配置简单，强烈推荐。

6).点击deploy now

7).看到他正在安装系统

8).安装完了，正在启动

9).开启成功

10).黄色字体忽略，是告诉你现在服务器刚开启成功~

11).等待片刻。如下图，我们找到ip和登陆密码即可使用mac自带的ssh工具登陆上服务器,如果出现ssh: connect to host slave port 22: Connection timed out，建议再换个IP。

12). 复制好ip地址 和密码，我们开始登陆到服务器上来。


代码如下：

root@服务器IP

13). 安装ss服务端


代码如下：

apt-get install shadowsocks

14).稍等片刻，服务器会自动安装好。然后我们输入如下命令，编辑下ss配置文件。


代码如下：

vim /etc/shadowsocks/config.json

看到如下初始配置

下面是我推荐的配置：

{
    "server":"这里写你的vultr服务器的ip",
    "server_port":8000,
    "local_address": "127.0.0.1",
    "local_port":1080,
    "password":"这里写你的ss密码，自己定义就行",
    "timeout":300,
    "method":"aes-256-cfb",
    "fast_open": false,
    "workers": 1
}

来看下mac 下如何在服务器上修改文件

下面是演示mac vultr搭建ss 的保存

按了ESC之后，会发现我们退出了插入模式，下面的INSERT消失了

15).开启ss服务器进程。


代码如下：

ssserver -c /etc/shadowsocks/config.json start &

16).服务器端开始运行了，我们开始下载使用客户端。

MAC 版连接
下载地址：

ShadowsocksX-2.6.3.dmg

然后 打开ss的 服务器配置，填入相关信息。

17).打开浏览器 访问网站，下载你需要的软件，enjoy yourself
