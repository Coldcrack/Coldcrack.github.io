使用idea创建一个springboot的web项目

[toc]

##前言
最近想看看java的东西，看到了微服务这个现在比较流行，他可以帮助程序员快速搭建一个后台服务，不用麻烦的去配置tomcat这些东西，应为springboot本身就内置tomcat。用它我们3分钟就可以搭建一个后台服务了。

创建web项目

首先打开idea

create new project

找到Spring Initializr,选择项目的jdk包，点击next。
这里默认服务url就是https://start.spring.io,这是创建springboot的官网，如果没有idea你也可以上官网创建一个springboot项目。

group就是随个人喜好，构建项目的类型type,这里有maven和gradle,看你环境装了那个用哪个。然后下一步

因为是web项目，这里直接选择web,版本号我这里选择1.5.20，点击next。

项目路径随意，点击finish。

这时会弹出一个上图，第一次构建项目是，gradle home是空的，我们选择Use local Gradle distribution 是为了把构建使用过的包都下载保存下来，方便以后编译更快一些，不用每次都从网上下载，路径是你下载安装gradle的路径。
点击OK，然后就是等待了,第一次比较慢，以后就很快了。

这里直接control+r运行就可以了。

出现

Started HelloWorldApplication in 1.491 seconds (JVM running for 1.936)

说明运行成功，打开浏览器输入localhost:8080

这就完成了。

配置文件

如果你觉得输端口号太烦，这里可以改，在目录中找到resource文件，
resource->application.properties

输入

server.port=80


端口号可以填其他的，范围是1到65535
